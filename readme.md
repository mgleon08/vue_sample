#Vue Sample

vue sample

#Install

* node.js
* npm or yarn
* webpack
* webpack-dev-server

#Usage

```
git clone git@github.com:mgleon08/vue_sample.git
npm install || yarn install
npm run dev || yarn run dev
open http://localhost:8080/
```
